# ❤️ Heart Disease Prediction App

A Machine Learning-based web application that predicts the likelihood of heart disease using user health data. This app is built with **Streamlit** and uses a trained **KNN model** for prediction.

---

## 🚀 Features

* Simple and interactive UI
* Real-time heart disease prediction
* Uses trained Machine Learning model
* Fast and lightweight

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy

---

## 📊 How It Works

1. User inputs health details (age, cholesterol, etc.)
2. Data is preprocessed using a scaler
3. Model predicts whether heart disease is likely or not
4. Result is displayed instantly on screen

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
